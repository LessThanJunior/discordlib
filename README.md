# Discord API C++ Library

## Dependencies:

- **Boost.Asio** - Uses with openssl
- **nlohmann/json**

## Implemented Features

- HTTPS GET requests
- Retrieve a Guild by id
- Retrieve a Channel by id
- Retrieve the current bot user
- Retrieve a User by id
- Retrieve Roles from a Guild
